Server Configuration

To make your settings persistent across all browsers and tours, export to a server configuration file.

Permanent Configuration: Export your settings to a JSON file that can be used for persistent configuration across all browsers and devices.

  1. Click "Export for Server" to download the configuration file
  2. Upload the file to /search-pro/config/search-config.json on your server
  3. The configuration will automatically load for all users

After setting up your permanent configuration, you can remove or restrict access to this settings page.
