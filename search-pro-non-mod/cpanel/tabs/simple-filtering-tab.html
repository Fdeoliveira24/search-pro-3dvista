Filter Mode

Choose how to filter content

Whitelist (Include)

Only show content that matches these terms

Enter terms separated by commas. Content containing any of these terms will be shown.

Blacklist (Exclude)

Hide content that matches these terms

Enter terms separated by commas. Content containing any of these terms will be hidden.

How It Works

Filter Examples:

  • Whitelist: If you enter "restaurant, hotel" - only hotspots/content labeled with "restaurant" or "hotel" will appear in search results
  • Blacklist: If you enter "private, restricted" - any content labeled with "private" or "restricted" will be hidden
  • Both: Use whitelist to include only certain types, then blacklist to exclude specific items from those types
